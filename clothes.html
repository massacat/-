تنسيق الألوان

اللون الأساسي الألوان المتناسقة معه الأفضل للخمار الحقيبة / الحذاء
أبيض كل الألوان أي لون قوي (أحمر، أزرق، أخضر) نفس لون الخمار أو لون محايد
أسود كل الألوان أي لون قوي أو باستيل نفس لون الخمار أو لون معدني (ذهبي/فضي)
بيج بني، أبيض، أخضر زيتوني، ذهبي بني أو أخضر درجات البيج أو البني
رمادي فاتح وردي، أزرق سماوي، بنفسجي باستيل وردي أو أزرق وردي أو فضي
أزرق سماوي أبيض، بيج، رمادي، أصفر باستيل أبيض أو بيج أبيض أو ذهبي
أزرق كحلي أبيض، بيج، أصفر، وردي أبيض أو وردي بني أو بيج
أخضر زمردي ذهبي، بيج، أبيض ذهبي أو بيج ذهبي أو بيج
أخضر زيتوني بيج، أبيض، بني بيج أو بني بني أو بيج
وردي باستيل رمادي فاتح، أبيض، بيج أبيض أو رمادي أبيض أو ذهبي
أحمر خمري أسود، بيج، ذهبي أسود أو بيج أسود أو ذهبي
بني كراميل بيج، أبيض، أخضر زيتوني بيج أو أخضر بيج أو بني
بنفسجي فاتح أبيض، رمادي، وردي أبيض أو وردي فضي أو أبيض
أصفر باستيل أبيض، بيج، رمادي فاتح أبيض أو بيج بيج أو أبيض
نصائح عامة:
- إذا كان الخمار بلون قوي، اجعلي الحقيبة أو الحذاء بنفس لونه، أو بلون محايد.
- إذا كانت الملابس ملونة أو مزخرفة، اجعلي الخمار والحقيبة والحذاء بلون واحد هادئ.
- إذا كانت الملابس والخمار بألوان هادئة، أبرزي الحقيبة أو الحذاء بلون لافت.
- الألوان المحايدة مثل الأبيض، الأسود، البيج، والرمادي تناسب جميع الألوان وتُستخدم كأساس متين للإطلالة.
- استخدام لون قوي في قطعة واحدة فقط يعطي مظهرًا أنيقًا ومتوازنًا.
- الألوان القوية تستخدم في القطع الصغيرة (حذاء. حقيبة. حزام) لابراز الإطلالة.
- لون محايد أساسي + لون قوي = إطلالة متناسقة.
- لون قوي في الإطلالة = باقي الاكسسوارات محايدة.
- إطلالة محايدة بالكامل = حقيبة أو حذاء بلون قوي.
